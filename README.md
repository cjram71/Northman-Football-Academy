# Northman Football Academy

A responsive single-page website concept for the year-round Northman Football Academy program for European American football players aged 14–17.

## Positioning

The Academy is intentionally presented as distinct from the broader Northman Football Camps brand:

- selective, year-round player development rather than an open short-term camp
- monthly rotating Nordic hubs
- individualized strength, speed and combine preparation
- film study and digital team meetings
- direct US high school and college coach mentorship
- a structured two-year pathway toward international competition and recruiting exposure

## Current build

The website is contained in `index.html` and uses semantic HTML, embedded CSS and lightweight vanilla JavaScript. It has no framework or build-step requirement.

Included:

- responsive navigation and mobile menu
- SEO metadata, Open Graph metadata and structured data
- accessible page structure and reduced-motion support
- program, monthly cycle, two-year roadmap, regional hubs, recruiting pathway, eligibility, pricing and FAQ sections
- application CTA connected to the existing Northman Academy registration page

## Before public launch

1. Register and connect `northmanfootballacademy.com`.
2. Enable GitHub Pages or connect the repository to a static host such as Cloudflare Pages, Netlify or Vercel.
3. Replace the temporary application link with a dedicated academy application workflow.
4. Add parent/guardian consent, privacy policy, membership terms, cancellation terms and safeguarding information.
5. Publish confirmed quarterly camp dates rather than sample rotation locations.
6. Add approved coach, player and partner photography to the repository instead of relying on the current remote Northman image.
7. Connect analytics, search-console verification and conversion tracking.
8. Add payment only after academy acceptance; minors should not enter a subscription without parent or guardian authorization.

## Suggested domain setup

- Primary: `northmanfootballacademy.com`
- Redirects: `.eu`, `.academy`, and common spelling variants when available
- Existing Northman Football Camps site should link directly to the new Academy domain
